# Jueguito-gacha
Por ahora no estoy 100% seguro de qué proyecto hacer, pero lo que más me convence es un pequeño juego estilo gacha. En él, recolectarás monedas (aún no decido cómo) que podrás gastar para obtener objetos en un gacha. Estos te darán ciertos buffs para conseguir más monedas y así repetir el ciclo. En el futuro añadiré más funciones; por ahora, solo se me ha ocurrido lo más básico
